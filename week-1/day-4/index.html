Main

What is a main?

the main tag specifies the main content of a document

there should be only one main element in a document

Article

What is an article?

the article tag defines an independent, self-contained content

such as a blog post, a magazine or newspaper article, or a comment

Section

What is a section?

the section tag defines a section in a document

such as chapters, headers, footers, or any other sections of the document

Aside

What is an aside?

the aside tag defines content "aside" from the content it is placed in.

This means aside content should be INDIRECTLY related to the surround content
