Lists

  • Creating an unordered list(ul) tag will give you bulleted lists by default
  • A fun short hand with HTML is when creating li tags you can type li*3 and hit TAB/ENTER which will prompt your VS code to generate 3 li tags
  • Give it a try! You can do this for all types of tags if you need multiple of them in a specific area
    • You can create nested lists by indenting a ul tag
    • And then generate an li tag
  1. We can also create ordered lists by using the ol tag!
  2. We still use the li(listed item) tag in the ol element
    1. ol tags can be nested as well!
    2. Try experimenting with nesting lists to see what you can create!

Article

What is an article?

the article tag defines an independent, self-contained content

such as a blog post, a magazine or newspaper article, or a comment

Section

What is a section?

the section tag defines a section in a document

such as chapters, headers, footers, or any other sections of the document

Aside

What is an aside?

the aside tag defines content "aside" from the content it is placed in.

This means aside content should be INDIRECTLY related to the surround content
